Autolint
========
Autolint watches your files for jslint-errors.

Installation
------------
    git clone git@github.com:magnars/autolint.git
    cd autolint/
    git submodule init
    git submodule update
    

Basic usage
-----------
Start with: autolint [path to js-files]

    autolint \*\*/\*.js
    autolint src/\*/\*.js test/\*/\*.js
